Welcome to Media Manager

Welcome! Media Manager is a comprehensive tool for organizing, managing, and tracking your media library. This help center will guide you through all the features and functionality.

Key Features

  • Multi-library support for organizing different media collections
  • Automatic metadata fetching from TMDB and TVDB
  • Advanced search and filtering capabilities
  • Batch operations for efficient media management
  • Tags, favorites, and collections for custom organization
  • Import and export functionality for backup and migration
  • Poster and subtitle downloading
  • NFO file generation for media players

Getting Started

If you're new to Media Manager, we recommend starting with the Quick Start Guide which will walk you through the basic setup and initial configuration.
