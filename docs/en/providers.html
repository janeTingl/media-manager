Metadata Providers

Media Manager fetches metadata from online databases to automatically fill in information about your media.

Supported Providers

TMDB (The Movie Database)

The primary provider for movie and TV show metadata.

  • Website: themoviedb.org
  • API Key: Get your API key
  • Free: Yes, for non-commercial use

TVDB (The TV Database)

Specialized database for TV shows and series.

  • Website: thetvdb.com
  • API Key: Get your API key
  • Free: Yes, for non-commercial use

Configuring Providers

  1. Open Edit â†’ Preferences
  2. Navigate to the Providers tab
  3. Enter your API keys
  4. Configure additional settings:
    • Enabled Providers: Choose which providers to use
    • Timeout: Network timeout in seconds
    • Retry Count: Number of retry attempts
  5. Click Apply
Important: You need at least one provider configured to fetch metadata. Without API keys, you'll need to enter all information manually.

Provider Priority

When multiple providers are enabled, Media Manager will:

  1. Try the first enabled provider
  2. If no results, try the next provider
  3. Use the best matching result

Caching

Provider responses are cached to improve performance and reduce API calls. You can configure caching in the Advanced preferences tab.
