Troubleshooting

Common issues and their solutions.

Metadata Issues

No metadata is being fetched

Solutions:
  • Verify your API keys are configured correctly in Edit → Preferences → Providers
  • Check your internet connection
  • Ensure at least one provider is enabled
  • Check the logs for error messages

Wrong metadata is matched

Solutions:
  • Use the search function to manually find the correct match
  • Rename your files to include the year (e.g., "Movie Title (2020).mkv")
  • Edit metadata manually after matching

Scanning Issues

Files are not being detected

Solutions:
  • Verify the file format is supported (mkv, mp4, avi, etc.)
  • Check that the files are in the library's root path
  • Ensure you have read permissions for the directory

Scanning is very slow

Solutions:
  • Large libraries take time on first scan
  • Subsequent scans will be faster due to caching
  • Consider splitting very large libraries

Performance Issues

Application is slow or unresponsive

Solutions:
  • Clear the cache in Edit → Preferences → Advanced
  • Reduce the number of enabled poster types
  • Increase cache TTL to reduce provider calls
  • Close other resource-intensive applications

UI Issues

Posters not displaying

Solutions:
  • Check poster download settings in Preferences
  • Verify cache directory is writable
  • Try clearing and re-downloading posters

Getting More Help

If your issue persists:

  • Check the application logs (usually in ~/.media-manager/logs/)
  • Review other help topics for detailed information
  • Report bugs to the development team
