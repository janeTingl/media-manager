Quick Start Guide

This guide will help you get started with Media Manager in just a few minutes.

Step 1: Create Your First Library

A library is a collection of media files. To create one:

  1. Click File → Manage Libraries from the menu
  2. Click Add Library
  3. Enter a name (e.g., "Movies" or "TV Shows")
  4. Select the media type (Movie or TV)
  5. Choose the folder containing your media files
  6. Click Save

Learn more: Library Setup

Step 2: Configure Metadata Providers

To automatically fetch information about your media, you need to configure API keys:

  1. Open Edit → Preferences
  2. Go to the Providers tab
  3. Enter your API keys for TMDB and/or TVDB
  4. Click Apply

Getting API keys:

  • TMDB: Visit themoviedb.org/settings/api
  • TVDB: Visit thetvdb.com/dashboard/account/apikeys

Learn more: Metadata Providers

Step 3: Scan Your Media

Once your library is set up:

  1. Select your library from the left panel
  2. Your media files will be automatically scanned
  3. The system will attempt to match files with online metadata
  4. Review and confirm matches in the Matching tab
  5. Click Finalize to save the metadata

Learn more: Scanning Media

Step 4: Explore Your Library

After scanning, you can:

  • Switch between Grid View and Table View
  • Use the Search tab to find specific items
  • Click on any item to view detailed information
  • Right-click for additional options like editing metadata or downloading posters
