Library Setup

Libraries are the foundation of Media Manager. Each library represents a collection of media files of a specific type.

Creating a Library

  1. Go to File â†’ Manage Libraries
  2. Click Add Library
  3. Fill in the library details:
    • Name: A descriptive name (e.g., "Action Movies", "TV Shows")
    • Type: Either Movie or TV
    • Root Path: The folder containing your media files
  4. Click Save
Tip: You can create multiple libraries for different collections. For example, separate libraries for "Kids Movies", "Adult Movies", "TV Shows", etc.

Organizing Your Files

For best results, organize your media files before adding them to a library:

Movies

Recommended structure:

/Movies/
  Movie Title (Year)/
    Movie Title (Year).mkv

TV Shows

Recommended structure:

/TV Shows/
  Show Name/
    Season 01/
      Show Name S01E01.mkv
      Show Name S01E02.mkv
Note: Media Manager can handle various naming conventions, but following these patterns will improve matching accuracy.

Managing Libraries

  • Edit: Select a library and click Edit to modify its settings
  • Delete: Select a library and click Delete to remove it (files are not deleted)
  • Set as Default: Right-click a library to set it as the default

Multiple Libraries

You can have multiple libraries for different purposes:

  • Separate movies and TV shows
  • Different quality levels (4K, HD, SD)
  • Different genres or categories
  • Personal vs. Family content
