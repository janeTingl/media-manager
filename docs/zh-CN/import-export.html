导入与导出

影藏·媒体管理器支持导入和导出媒体库数据，便于备份和迁移。

导出媒体

选择 编辑 → 导出媒体 或按 Ctrl+E。

导出格式

  • JSON 格式： 完整的媒体库数据，包含所有元数据
  • NFO 文件： Kodi/Jellyfin 兼容的元数据文件
  • CSV 格式： 表格数据，可用 Excel 打开

导出选项

  • 选择要导出的媒体库
  • 包含/排除特定字段
  • 导出海报和图片
  • 选择导出位置

导入媒体

选择 编辑 → 导入媒体 或按 Ctrl+I。

支持的导入格式

  • JSON 格式： 从之前的导出文件恢复
  • NFO 文件： 从 Kodi/Jellyfin 导入元数据
  • CSV 格式： 从电子表格批量导入

导入选项

  • 合并或替换现有数据
  • 处理重复项
  • 验证导入的数据

备份策略

建议定期备份媒体库：

  • 每月导出完整的媒体库数据
  • 在重大更改前创建备份
  • 将备份文件存储在安全位置
