快速入门指南

按照以下步骤操作，只需几分钟就能完成 影藏·媒体管理器 的基本设置。

步骤 1：建立第一个媒体库

媒体库是档案的集合，要创建一个媒体库：

  1. 在菜单中选择 File → Manage Libraries
  2. 点击 Add Library
  3. 输入一个名称（例如「Movies」或「TV Shows」）
  4. 选择媒体类型（Movie 或 TV）
  5. 指定存放媒体档案的资料夹
  6. 点击 Save

延伸阅读：媒体库设置

步骤 2：配置元数据提供商

若要自动取得影片资讯，需要先填写 API 金钥：

  1. 打开 Edit → Preferences
  2. 切换到 Providers 分页
  3. 输入 TMDB 与/或 TVDB 的 API 金钥
  4. 点击 Apply

如何取得 API 金钥：

  • TMDB：造访 themoviedb.org/settings/api
  • TVDB：造访 thetvdb.com/dashboard/account/apikeys

延伸阅读：元数据提供商

步骤 3：扫描媒体档案

完成媒体库后：

  1. 在左侧面板选择要扫描的媒体库
  2. 系统会自动检测资料夹中的档案
  3. 应用程序会尝试与线上资料库匹配
  4. 在 Matching 分页中审查并确认结果
  5. 点击 Finalize 保存元数据

延伸阅读：扫描媒体

步骤 4：浏览你的收藏

扫描完成后，你可以：

  • 切换 Grid View 与 Table View
  • 使用 Search 分页快速找到项目
  • 点击任何条目查看详细资讯
  • 按右键可编辑元数据或下载海报

下一步

熟悉基本流程后，可以进一步探索：

  • 自定义元数据
  • 对多个项目执行批量操作
  • 利用标签与收藏整理内容
  • 备份或汇出资料
