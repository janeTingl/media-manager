快速上手指南

只需几分钟即可完成 Media Manager 的基本配置。按照以下四个步骤完成初始设置：

步骤 1：创建媒体库

媒体库用于组织影片或剧集：

  1. 点击菜单 文件 → 管理媒体库
  2. 选择 添加媒体库
  3. 输入名称（例如“电影”或“剧集”）
  4. 选择媒体类型（电影 / 电视剧）
  5. 指定存放视频文件的根目录
  6. 点击 保存

更多说明：媒体库设置

步骤 2：配置元数据提供者

输入 TMDB / TVDB API Key 以自动拉取影片信息：

  1. 打开 编辑 → 偏好设置
  2. 切换到 提供者 分页
  3. 填写 TMDB 与 / 或 TVDB 的 API Key
  4. 点击 应用

获取密钥：

  • TMDB：themoviedb.org/settings/api
  • TVDB：thetvdb.com/dashboard/account/apikeys

步骤 3：扫描媒体

创建媒体库后，程序会自动扫描文件夹：

  1. 在左侧面板选择目标媒体库
  2. 系统会识别并列出所有视频文件
  3. 匹配结果将在 匹配 标签页中展示
  4. 核对信息并点击 Finalize 完成写入

步骤 4：浏览与管理

扫描完成后你可以：

  • 在 网格视图 与 表格视图 之间切换
  • 使用 搜索 标签快速定位内容
  • 右键条目以编辑元数据、下载海报或添加标签
