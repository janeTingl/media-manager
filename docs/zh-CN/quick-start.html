快速入门指南

按照以下步骤操作，只需几分钟就能完成影藏·媒体管理器的基本设置。

步骤 1：建立第一个媒体库

媒体库是媒体文件的集合，要创建一个媒体库：

  1. 在菜单中选择 文件 → 管理媒体库
  2. 点击 添加媒体库 按钮
  3. 输入一个名称（例如「电影」或「电视剧」）
  4. 选择媒体类型（电影 或 电视剧）
  5. 指定存放媒体文件的文件夹
  6. 点击 保存

延伸阅读：媒体库设置

步骤 2：配置元数据提供商

若要自动获取影片信息，需要先填写 API 密钥：

  1. 打开 编辑 → 首选项
  2. 切换到 提供商 标签页
  3. 输入 TMDB 和/或 TVDB 的 API 密钥
  4. 点击 应用

如何获取 API 密钥：

  • TMDB：访问 themoviedb.org/settings/api
  • TVDB：访问 thetvdb.com/dashboard/account/apikeys

延伸阅读：元数据提供商

步骤 3：扫描媒体文件

创建媒体库后：

  1. 在左侧面板选择要扫描的媒体库
  2. 系统会自动检测文件夹中的文件
  3. 应用程序会尝试与在线数据库匹配
  4. 在 匹配 标签页中审查并确认结果
  5. 点击 完成匹配 保存元数据

延伸阅读：扫描媒体

步骤 4：浏览你的收藏

扫描完成后，你可以：

  • 切换 网格视图 与 表格视图
  • 使用 搜索 标签页快速找到项目
  • 点击任何条目查看详细信息
  • 右键点击可编辑元数据或下载海报

下一步

熟悉基本流程后，可以进一步探索：

  • 自定义元数据
  • 对多个项目执行批量操作
  • 利用标签与收藏整理内容
  • 备份或导出数据
