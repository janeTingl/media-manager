元数据提供商

影藏·媒体管理器 会从线上数据库抓取影片资讯，并自动填入标题、年份、海报等细节。

支援的提供商

TMDB（The Movie Database）

主要的电影与影集资料来源。

  • 网站： themoviedb.org
  • API 金钥： 申请金钥
  • 费用： 非商业用途免费

TVDB（The TV Database）

专注于电视与影集的资料库。

  • 网站： thetvdb.com
  • API 金钥： 申请金钥
  • 费用： 非商业用途免费

设定提供商

  1. 打开 Edit → Preferences
  2. 切换到 Providers 页签
  3. 输入各提供商的 API 金钥
  4. 如有需要，可进一步设定：
    • Enabled Providers： 要启用的来源
    • Timeout： 网络请求逾时时间（秒）
    • Retry Count： 失败时的重试次数
  5. 点击 Apply
重要： 至少需要启用一个提供商才能抓取资料；若没有 API 金钥，所有资讯都得手动输入。

提供商优先顺序

当启用多个提供商时，影藏·媒体管理器 会依顺序尝试：

  1. 先使用第一个启用的提供商
  2. 若没有结果，再尝试下一位
  3. 采纳匹配度最高的结果

快取

为了提升速度与减少 API 呼叫，提供商响应会被快取。相关设定可在 Advanced 分页调整。
