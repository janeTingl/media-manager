元数据提供者

Media Manager 借助在线数据库自动补全影片信息。本页面介绍如何配置与管理提供者。

支持的提供者

TMDB（The Movie Database）

主要用于电影与电视剧的元数据。

  • 官网： themoviedb.org
  • API Key： 申请地址
  • 授权： 非商业用途免费

TVDB（The TV Database）

专注电视剧与季、集信息。

  • 官网： thetvdb.com
  • API Key： 申请地址
  • 授权： 非商业用途免费

配置步骤

  1. 打开 编辑 → 偏好设置
  2. 切换到 提供者 标签
  3. 输入 API Key
  4. 根据需要调整：
    • 启用的提供者： 选择要使用的数据源
    • 超时时间： 调整网络请求时限
    • 重试次数： 设置失败后的重试次数
  5. 点击 应用
重要： 至少需要启用一个提供者才能自动抓取元数据，否则必须手动输入所有信息。

优先顺序与缓存

启用多个提供者时，系统会依照列表顺序依次查询，直到找到最佳结果。所有响应都会缓存在本地，你可以在 高级 标签中调整缓存路径与有效期。
