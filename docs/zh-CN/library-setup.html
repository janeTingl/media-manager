媒体库设置

媒体库是 Media Manager 的核心，每个库都代表一组具有相同类型或用途的媒体文件。

创建媒体库

  1. 打开 文件 → 管理媒体库
  2. 点击 添加媒体库
  3. 填写以下信息：
    • 名称： 建议使用描述性名称（如“动作片”或“家庭剧集”）
    • 类型： 电影或电视剧
    • 根目录： 存放媒体文件的上级文件夹
  4. 点击 保存
提示： 你可以为不同场景建立多个媒体库，例如“动漫”、“纪录片”、“儿童影片”等。

推荐的文件结构

电影

/Movies/
  电影名称 (年份)/
    电影名称 (年份).mkv

电视剧

/TV Shows/
  剧名/
    Season 01/
      剧名 S01E01.mkv
      剧名 S01E02.mkv
注意： 程序能识别多种命名规则，但保持如上结构可显著提升匹配准确率。

管理媒体库

  • 编辑： 选择媒体库后点击 编辑 修改名称或路径
  • 删除： 删除媒体库不会移除硬盘上的实际文件
  • 设为默认： 右键媒体库可设为默认加载

多媒体库场景

根据需要，你可以将电影与剧集、不同清晰度或不同成员的收藏拆分到独立媒体库中。这能让扫描与筛选更加清晰。
