媒体库设置

媒体库是 影藏·媒体管理器 的基础。每个媒体库都代表一组特定类型的媒体档案。

如何创建媒体库

  1. 打开 File → Manage Libraries
  2. 点击 Add Library
  3. 填写以下资讯：
    • Name： 描述性名称（例："Action Movies"、"TV Shows"）
    • Type： Movie 或 TV
    • Root Path： 实际存放媒体档案的资料夹
  4. 点击 Save
提示： 可以为不同的收藏建立多个媒体库，例如「儿童电影」「纪录片」或依照画质区分等。

整理档案结构

在建立媒体库前先整理档案可提升匹配准确度：

电影

推荐结构：

/Movies/
  Movie Title (Year)/
    Movie Title (Year).mkv

影集

推荐结构：

/TV Shows/
  Show Name/
    Season 01/
      Show Name S01E01.mkv
      Show Name S01E02.mkv
注意： 影藏·媒体管理器 能处理多种命名方式，但遵循上述结构有助于提升自动匹配成功率。

管理媒体库

  • Edit： 选中媒体库后点击 Edit 可修改设定
  • Delete： 移除媒体库（不会删除磁碟上的档案）
  • Set as Default： 右键媒体库可设为默认

多媒体库策略

依用途建立多个媒体库：

  • 将电影与影集分开
  • 依画质（4K、HD、SD）分类
  • 依题材或分类组织
  • 划分个人内容与家庭内容
