欢迎使用 Media Manager

欢迎体验 Media Manager！这款桌面工具可以帮助你整理、管理并跟踪整个媒体库。本帮助中心提供了所有主要功能的中文说明，方便你快速上手。

核心特性

  • 支持多个媒体库，轻松区分不同的收藏
  • 自动从 TMDB 与 TVDB 抓取影片与剧集信息
  • 提供强大的搜索、过滤与排序方式
  • 批量操作可一次处理大量条目
  • 使用标签与收藏夹快速归类内容
  • 提供导入 / 导出与备份支持
  • 可下载海报与字幕，并自定义保存策略
  • 生成兼容播放器的 NFO 元数据文件

快速入门

如果你是第一次使用，建议从 快速上手指南 开始，按照向导完成媒体库与提供者的基础配置。

需要帮助？

左侧列表列出了所有主题，你也可以使用搜索框输入关键字来查找想要的内容。按下 F1 可随时打开与当前界面相关的帮助页面。
