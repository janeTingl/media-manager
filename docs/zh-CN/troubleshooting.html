故障排除

以下列出常见问题与对应的解决方式。

元数据问题

无法取得任何元数据

解决方法：
  • 确认已在 编辑 → 首选项 → 提供商 中填写正确的 API 密钥
  • 检查网络连接是否正常
  • 确定至少启用一个提供商
  • 查看日志是否有错误信息

匹配结果完全不正确

解决方法：
  • 使用搜寻功能手动挑选正确的结果
  • 将档名加入年份（例如「Movie Title (2020).mkv」）
  • 在匹配后手动编辑元数据

扫描问题

找不到任何档案

解决方法：
  • 确认档案格式受支援（mkv、mp4、avi 等）
  • 确保档案位于媒体库的 root path 内
  • 检查是否具有读取该资料夹的权限

扫描速度非常慢

解决方法：
  • 第一次处理大型媒体库时需要较久时间
  • 后续扫描会因为快取而更快
  • 可考虑拆分为多个较小的媒体库

效能问题

程序反应迟缓或冻结

解决方法：
  • 在 编辑 → 首选项 → 高级 中清除缓存
  • 减少一次下载的海报类型
  • 提高缓存 TTL 以降低 API 请求频率
  • 关闭其他占用资源的应用程序

界面问题

海报没有显示

解决方法：
  • 检查首选项里的海报下载设定
  • 确认快取资料夹可写入
  • 尝试清除并重新下载海报

需要更多帮助？

若问题仍然存在，可以：

  • 查看日志（一般位于 ~/.media-manager/logs/）
  • 查阅其他说明主题以获得更完整资讯
  • 向开发团队回报错误
