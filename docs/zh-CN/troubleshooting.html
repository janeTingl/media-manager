故障排除

遇到问题时，可从以下检查开始。

常见问题

  • 无法加载媒体库： 确认根目录仍然存在且具有读取权限。
  • 匹配失败： 检查 TMDB / TVDB API Key 是否有效，或尝试手动搜索。
  • 海报 / 字幕下载失败： 查看网络连接，并在 下载 设置中延长超时。
  • 程序未显示中文界面： 在 偏好设置 → UI 中选择 简体中文 或 繁体中文，然后重启程序。

日志文件

日志位于 ~/.media-manager/logs/app.log。若需要向支持团队反馈，请附上最新日志。

提示： 若界面出现异常，可在 偏好设置 → UI 取消“记住房屋布局”，然后重新启动应用。

进一步支持

若以上步骤无法解决问题，请在 帮助 → 报告问题 中提供详细描述与截图，我们会尽快跟进。
