标签与收藏

使用标签和收藏功能可以更好地组织和管理您的媒体收藏。

标签

标签是自定义的分类标记，可以为媒体项目添加多个标签。

添加标签

  1. 选择一个或多个媒体项目
  2. 右键点击并选择 添加标签
  3. 输入标签名称
  4. 或从现有标签中选择

管理标签

  • 在元数据编辑器中查看和编辑标签
  • 使用批量操作为多个项目添加/删除标签
  • 按标签筛选媒体库中的项目

常用标签示例

  • 类型标签：科幻、动作、喜剧、恐怖
  • 观看状态：已观看、待观看、重看
  • 质量标签：4K、1080p、720p
  • 收藏类别：经典、奥斯卡、漫威

收藏

将喜爱的媒体项目标记为收藏，便于快速访问。

添加到收藏

  • 选择项目后点击收藏按钮（星形图标）
  • 右键点击并选择 添加到收藏
  • 在元数据编辑器中勾选收藏选项

查看收藏

  • 点击主窗口的 收藏 标签页
  • 在搜索中使用收藏筛选器
  • 在媒体库视图中按收藏状态排序

最佳实践

  • 使用一致的标签命名方案
  • 定期整理和清理标签
  • 为不同用途创建标签类别
  • 使用收藏功能突出显示精选内容
