元数据提供商

Media Manager 透过线上数据库取得影片与剧集资讯。配置有效的提供商可以大幅减少手动输入。

可用的提供商

TMDB (The Movie Database)

主要用于电影与剧集的基础资讯。

  • 网站： themoviedb.org
  • API Key： 前往申请
  • 使用许可： 非商业用途免费

TVDB (The TV Database)

专注于剧集、动画与综艺等内容。

  • 网站： thetvdb.com
  • API Key： 前往申请
  • 使用许可： 非商业用途免费

配置步骤

  1. 打开 编辑 → 偏好设置
  2. 切换到 提供商分页
  3. 输入各个服务的 API Key
  4. 确认以下选项：
    • 启用的提供商： 勾选需要使用的来源
    • 逾时： 联机逾时时间（秒）
    • 重试次数： 请求失败时的重新尝试次数
  5. 点击 套用
重要： 至少需启用一个提供商才能自动取得资料。若没有 API Key，只能手动编辑所有资讯。

优先顺序与快取

当多个提供商同时启用时，Media Manager 会依勾选顺序逐一查询，直到找到最合适的结果。回传资料会写入快取，以减少 API 调用并提升速度，可在 进阶 分页调整快取设定。
