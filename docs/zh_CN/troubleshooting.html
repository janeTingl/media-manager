疑难解答

以下列出最常见的问题与处理方式。

元数据相关

无法取得任何元数据

可能的解决方法：
  • 确认在 编辑 → 偏好设置 → 提供商 中已正确填入 API Key
  • 检查网路连接是否正常
  • 确保至少启用一个提供商
  • 打开日志查看是否有错误讯息

匹配到错误的作品

  • 在匹配视窗中手动搜索正确的条目再指定
  • 在档名加入年份以提高辨识度（例如《电影名 (2020).mkv》）
  • 匹配完成后可手动编辑并覆盖字段

扫描相关

文件未被侦测

  • 确认档案格式为常见的视频格式（mkv、mp4、avi 等）
  • 检查文件是否确实置于资料库的根目录或其子目录
  • 确认资料夹权限允许 Media Manager 读取

扫描速度非常慢

  • 第一次扫描大型资料库需要较长时间，后续扫描会因快取而加速
  • 可考虑将超大型的收藏拆成多个资料库

效能与介面

程式反应迟缓

  • 在 编辑 → 偏好设置 → 进阶 中清除快取再试
  • 减少需下载的海报尺寸或数量
  • 增加快取保存时间，降低对 API 的即时请求
  • 关闭其他占用大量资源的应用程式

海报不显示

  • 确认偏好设置中的海报下载选项已启用
  • 检查快取目录是否具备写入权限
  • 尝试清空并重新下载海报

取得更多协助

若问题仍存在：

  • 查看日志（默认位于 ~/.media-manager/logs/）
  • 阅读其他帮助主题，了解详细的操作说明
  • 将重现步骤提交给维护团队
