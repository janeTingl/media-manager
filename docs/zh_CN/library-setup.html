资料库设置

资料库是 Media Manager 的核心。每个资料库对应一组特定类型（电影或剧集）的文件夹。

如何建立资料库

  1. 打开 文件 → 管理资料库
  2. 点击 新增资料库
  3. 填写下列栏位：
    • 名称： 建议使用描述性名称，例如「动作电影」或「Netflix 剧集」
    • 类型： 选择 电影 或 剧集
    • 根路径： 指向存放媒体文件的资料夹
  4. 按下 储存
技巧： 你可以为不同来源、画质或家庭成员建立多个资料库，方便独立管理与筛选。

推荐的资料夹结构

电影

/Movies/
  电影名称 (年份)/
    电影名称 (年份).mkv

剧集

/TV Shows/
  剧名/
    Season 01/
      剧名 S01E01.mkv
      剧名 S01E02.mkv
提醒： Media Manager 能容忍大多数命名方式，但按照以上模式可显著提升自动匹配的准确度。

管理既有资料库

  • 编辑： 选取资料库后点击 编辑 可修改名称、类型或路径
  • 删除： 删除只会移除索引，不会删除磁碟上的原始文件
  • 设为预设： 右键资料库即可快速将其设为应用程式启动时加载的目标
