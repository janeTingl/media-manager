快速上手指南

按照以下 4 个步骤，你可以在几分钟内完成 Media Manager 的初始配置。

步骤 1：建立第一个资料库

资料库是媒体文件的集合，通常按照类型或存储位置区分。

  1. 打开菜单 文件 → 管理资料库
  2. 点击 新增资料库
  3. 填写名称（如「电影」、「动漫」）
  4. 选择类型（电影或剧集）
  5. 指定媒体所在的根目录
  6. 按下 储存

详细说明：资料库设置

步骤 2：配置元数据提供商

为了自动抓取影片资料，需要在偏好设置中填写 API Key：

  1. 打开 编辑 → 偏好设置
  2. 切换到 提供商 分页
  3. 填写 TMDB 与/或 TVDB 的 API Key
  4. 点击 套用

如何取得 API Key：

  • TMDB：themoviedb.org/settings/api
  • TVDB：thetvdb.com/dashboard/account/apikeys

延伸阅读：元数据提供商

步骤 3：扫描媒体文件

建立资料库后，系统会自动侦测并扫描文件：

  1. 在左侧选择刚建立的资料库
  2. Media Manager 会立即开始扫描
  3. 在 匹配 分页中确认自动配对的结果
  4. 必要时手动搜索正确的条目
  5. 点击 完成 保存元数据

延伸阅读：媒体扫描

步骤 4：探索你的资料库

扫描完成后，你可以：

  • 在 缩图 与 表格 视图之间切换
  • 利用 搜索 分页找到特定作品
  • 点选任一条目查看详细资讯、下载海报或编辑元数据
  • 透过批量操作一次调整多部影片
