关于 Media Manager

Media Manager 是一款以 PySide6 打造的本地应用程式，专注于让电影与剧集收藏保持干净、有序并具备美观的视觉呈现。

作者 janeT

janeT 是本软件的作者与产品负责人。她将个人在影音资料库管理上的经验转化成一套实用的流程——包含批量工具、仪表板、以及你现在看到的多语言帮助系统。

产品理念

  • 提供多资料库支持，适应不同硬盘或家庭成员的需求
  • 以自动化为核心：扫描、匹配、下载海报与字幕皆可一键完成
  • 保留人工控制权：随时可手动编辑、批量覆写或覆核记录
  • 强调可移植性与备份，方便在 NAS、HTPC 或工作站之间同步
  • 通过本地化界面与文档，让团队以最熟悉的语言协作
