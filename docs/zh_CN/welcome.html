欢迎使用 Media Manager

欢迎来到 Media Manager！这是一款专为整理电影与剧集而设计的桌面工具。通过本帮助中心，你可以快速了解每个界面、理解工作流程，并在需要时随时查阅参考信息。

核心功能

  • 支持多个资料库，轻松拆分不同的收藏或硬盘
  • 自动从 TMDB / TVDB 获取影片与剧集的详细资料
  • 强大的搜索与过滤器，帮助你在庞大库中秒找条目
  • 批量操作、标签与收藏夹让整理效率倍增
  • 内建海报、字幕与 NFO 下载流程，方便与播放器同步
  • 提供导入 / 导出功能，便于备份或迁移资料库

快速开始

第一次使用？建议从《快速上手指南》开始，按照步骤创建资料库、配置提供商，并完成首次扫描。

需要帮助？

左侧列表列出了所有主题，使用顶部的搜索框即可按关键字过滤。任何界面按下 F1 都会打开与当前视图相关的帮助页面。
