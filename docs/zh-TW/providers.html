中繼資料提供者

Media Manager 透過線上資料庫自動補齊影片資訊。本頁說明如何設定與管理提供者。

支援的提供者

TMDB（The Movie Database）

主要來源，適用於電影與影集的詳細資訊。

  • 網站： themoviedb.org
  • API 金鑰： 申請連結
  • 授權： 非商業用途免費

TVDB（The TV Database）

專門針對影集與季 / 集資訊。

  • 網站： thetvdb.com
  • API 金鑰： 申請連結
  • 授權： 非商業用途免費

設定步驟

  1. 開啟 編輯 → 偏好設定
  2. 切換到 提供者 分頁
  3. 輸入 API 金鑰
  4. 調整其他選項：
    • 啟用的提供者： 選擇要使用的資料來源
    • 逾時時間： 設定網路請求的秒數
    • 重試次數： 設定失敗後的重試次數
  5. 按下 套用
重要： 若未設定任何提供者，系統無法自動取得資料，必須手動輸入全部資訊。

優先順序與快取

當同時啟用多個提供者時，系統會依照列表順序逐一查詢，直到取得符合的結果。所有回應都會快取在本機，可在 進階 分頁調整快取路徑與有效期限。
