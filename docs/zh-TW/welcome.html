歡迎使用 Media Manager

感謝選擇 Media Manager！這是一款桌面應用程式，能協助你整理、管理並追蹤整個媒體收藏。此說明中心提供主要功能的繁體中文指南，協助你快速熟悉流程。

核心特色

  • 支援多個媒體庫，輕鬆區分不同收藏
  • 自動從 TMDB 與 TVDB 抓取影片與影集資訊
  • 提供進階搜尋、篩選與排序能力
  • 批次作業一次處理大量項目
  • 以標籤與最愛整理內容
  • 具備匯入 / 匯出與備份功能
  • 可下載海報與字幕並自訂儲存規則
  • 生成相容播放器的 NFO 中繼資料檔

快速開始

若你是第一次使用，建議先閱讀 快速上手指南，依照步驟完成媒體庫與提供者設定。

需要協助？

左側清單列出所有主題，也可使用上方搜尋框輸入關鍵字。按下 F1 可立即開啟與目前畫面相對應的說明頁面。

熱門主題

  • 建立第一個媒體庫
  • 設定 TMDB / TVDB API
  • 掃描並比對媒體檔案
  • 編輯與校正中繼資料
  • 解決常見問題
