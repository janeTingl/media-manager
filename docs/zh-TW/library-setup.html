媒體庫設定

媒體庫是 Media Manager 的基礎，每個媒體庫代表一組性質相同的影片或影集。

建立媒體庫

  1. 開啟 檔案 → 管理媒體庫
  2. 按下 新增媒體庫
  3. 填寫以下欄位：
    • 名稱： 建議使用清楚的描述字眼（如「動畫片」或「影集」）
    • 類型： 電影或影集
    • 根目錄： 儲存媒體檔案的資料夾
  4. 按下 儲存
小技巧： 可依照家庭成員、影片等級或影音格式建立多個媒體庫，以便管理。

建議的資料夾結構

電影

/Movies/
  電影名稱 (年份)/
    電影名稱 (年份).mkv

影集

/TV Shows/
  劇名/
    Season 01/
      劇名 S01E01.mkv
      劇名 S01E02.mkv
提醒： 系統可處理不同命名方式，但保持一致的資料夾層級有助於提升比對成功率。

管理媒體庫

  • 編輯： 選取媒體庫後點選 編輯 即可修改設定
  • 刪除： 刪除媒體庫並不會移除磁碟上的檔案
  • 設為預設： 在媒體庫上按右鍵可設為預設載入

多媒體庫使用情境

你可以將電影與影集或不同語言 / 畫質分開管理，讓掃描與篩選更加清楚。
