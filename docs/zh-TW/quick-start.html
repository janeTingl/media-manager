快速上手指南

依照下列四個步驟即可完成 Media Manager 的初始設定：

步驟 1：建立媒體庫

媒體庫用於區分不同類型的收藏：

  1. 點選選單 檔案 → 管理媒體庫
  2. 按下 新增媒體庫
  3. 輸入名稱（例如「電影」或「影集」）
  4. 選擇媒體類型（電影 / 影集）
  5. 指定存放影音檔案的資料夾
  6. 按 儲存

延伸閱讀：媒體庫設定

步驟 2：設定中繼資料提供者

輸入 TMDB / TVDB API Key，即可自動擷取節目資訊：

  1. 開啟 編輯 → 偏好設定
  2. 切換到 提供者 分頁
  3. 填入 TMDB 及 / 或 TVDB 的 API Key
  4. 按下 套用

取得金鑰：

  • TMDB：themoviedb.org/settings/api
  • TVDB：thetvdb.com/dashboard/account/apikeys

步驟 3：掃描媒體

媒體庫建立後系統會自動開始掃描：

  1. 在左側面板選擇媒體庫
  2. 程式會偵測並列出所有影片檔案
  3. 比對結果顯示在 比對 分頁
  4. 檢查資訊後按下 Finalize 儲存

步驟 4：瀏覽與管理

完成掃描後可以：

  • 在 網格檢視 與 表格檢視 間切換
  • 透過 搜尋 分頁快速找到條目
  • 右鍵點擊條目以編輯中繼資料、下載海報或新增標籤
